 # check-proxy
 
 Async proxy validator with optional public proxy fetching.
 
 ## Structure
 
 - `src/`: scanner, fetcher, CLI entry
 - `data/`: sample input lists
 - `scripts/`: helper scripts for generating test inputs
 - `docs/`: usage documentation
 
 ## Docs
 
 See `docs/HOWTO.md` for installation and usage.
